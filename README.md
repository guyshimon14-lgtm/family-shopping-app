# Family Shopping App

אפליקציית MVP לניהול רשימת קניות משפחתית בעברית.

## מה יש עכשיו

- הוספת פריטים עם כמות, קטגוריה והערה/חנות.
- סימון פריטים כנקנו.
- סינון לפי הכול / פתוחים / נקנו.
- ניקוי פריטים שנקנו.
- שיתוף הרשימה כטקסט.
- שמירה מקומית בדפדפן באמצעות `localStorage`.

## איך מריצים מקומית

אין צורך בהתקנת חבילות. אפשר לפתוח את `index.html` בדפדפן, או להריץ שרת סטטי פשוט:

```bash
python3 -m http.server 5173
```

ואז לפתוח:

```text
http://localhost:5173
```

## פריסה ל-Netlify

הריפו כולל `netlify.toml` שמגדיר פריסה סטטית מהשורש:

```toml
[build]
  publish = "."
```

ב-Netlify בוחרים Import from Git, מחברים את GitHub, בוחרים את הריפו `family-shopping-app`, ואת branch `main`.

## מגבלות MVP

כרגע הרשימה נשמרת רק בדפדפן של המשתמש. כדי להפוך אותה לרשימה משפחתית אמיתית שמסתנכרנת בין כמה מכשירים, השלב הבא יהיה להוסיף backend קל, למשל Supabase/Firebase/Netlify Blobs.
