# Family App v1

גרסה ראשונה נקייה לאפליקציית משפחה לאייפון.

## מה יש עכשיו

- שתי לשוניות: קניות ותחזוקה.
- שלושה משתמשים קבועים: אני, אשתי, הילדה.
- רק "אני" מוגדר כמנהל.
- הוספת מוצר בקול או בהקלדה.
- זיהוי כמות ומוצר בעברית, למשל: `2 מלפפונים`.
- איחוד כמויות לאותו מוצר אחרי אישור המשתמש.
- התרעה כאשר מוצר כבר קיים ברשימה.
- קיבוץ מוצרים לפי קטגוריות.
- פקודת מנהל לסגירת הרשימה עד יום ושעה, למשל: `סגור את הרשימה עד יום חמישי בשעה 20`.
- התאמה למסך אייפון והוספה למסך הבית דרך Safari.
- הכנה כ-PWA: קובץ manifest, service worker, וקבצי Netlify בסיסיים.

## איך מתקינים באייפון אחרי פרסום

1. פותחים את כתובת האפליקציה ב-Safari באייפון.
2. לוחצים על כפתור השיתוף.
3. בוחרים "הוסף למסך הבית".
4. מאשרים את השם Family.
5. פותחים מהאייקון החדש במסך הבית.

## פרסום ל-Netlify

אפשר לפרסם את התיקייה הזאת כמו שהיא:

`outputs/family-app-v1`

הקובץ `netlify.toml` כבר מגדיר את תיקיית הפרסום, כותרות פרטיות בסיסיות, והגדרה נכונה ל-service worker.

## חשוב לדעת

בשלב הזה הנתונים עדיין נשמרים בדפדפן של כל מכשיר. זה טוב לבדיקה, אבל לא מספיק לאפליקציה משפחתית אמיתית.

כדי שהאפליקציה תהיה באמת פתוחה רק לשלושתכם, תסתנכרן בין האייפונים, ותשלח פוש אמיתי, צריך לחבר:

- התחברות מאובטחת לשלושה משתמשים בלבד.
- בסיס נתונים בענן, למשל Firebase Firestore או Supabase.
- הרשאות משתמשים: מנהל מול שותפות.
- Web Push דרך service worker.
- פרסום לכתובת HTTPS.
